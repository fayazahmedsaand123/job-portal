# Job Portal - Laravel + Tailwind CSS

A modern Job Portal built with Laravel and Tailwind CSS.

## Features
- Role based authentication (Admin, Employer, Candidate)
- Employer can post jobs and manage applicants
- Candidate can browse jobs and apply with resume
- Purple Admin Panel with charts and real-time search
- Beautiful home page with multiple sections

## Tech Stack
- Laravel 9
- MySQL
- Tailwind CSS
- Chart.js
- JavaScript / Ajax

## How to Run
1. Clone the repository
2. Run `composer install`
3. Copy `.env.example` to `.env`
4. Set database credentials in `.env`
5. Run `php artisan migrate`
6. Run `php artisan serve`
